PHP library for Business Messages API v1

Installation

Python 2.7 is required.  Python3 is not supported.

Everything can be installed quickly with easy_install or pip. E.g.:

  pip install google-apis-client-generator

generate_library --input=bm.json --language=php --output_dir=c:/tmp/generated

https://businessmessages.googleapis.com/$discovery/rest?version=v1

Requirements:

Google API PHP Client, version 1.1.1
URL: https://github.com/google/google-api-php-client/releases
